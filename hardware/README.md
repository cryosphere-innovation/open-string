# Hardware

<!-- SPDX-License-Identifier: CC-BY-SA-4.0 -->

KiCad project and fabrication outputs for the OpenString PCB and cable
assembly.

## Contents

- `board-files/` — KiCad project (schematic, PCB layout, project settings)
- `fabrication/gerbers/` — Gerbers and drill files ready to upload to a PCB house

## Ordering

See [`docs/ordering.mdx`](../docs/ordering.mdx) for the current ordering
guide (fab house settings, stencil, assembly options, expected cost).
