# Firmware

<!-- SPDX-License-Identifier: CC-BY-SA-4.0 -->

Arduino firmware for the OpenString microcontroller.

## Building

1. Install the [Arduino IDE](https://www.arduino.cc/en/software) (or
   `arduino-cli`).
2. Install the required board support package and libraries listed in
   `libraries.txt`.
3. Open `open_string/open_string.ino`.
4. Select the correct board and port, then Upload.

See [`../docs/firmware.mdx`](../docs/firmware.mdx) for the full flashing
guide, including fuse settings and troubleshooting.
