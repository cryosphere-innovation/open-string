# Ordering

This page walks through ordering everything you need to build an
OpenString from scratch.

## 1. PCB

Fabrication files live in [`hardware/gerbers/`](https://github.com/cameronplanck/open-string/tree/main/hardware/gerbers).

Recommended fab houses:

- **JLCPCB** — cheapest for small runs, assembly available
- **PCBWay** — similar pricing, good QA
- **OSH Park** — US-based, higher quality, longer lead time
