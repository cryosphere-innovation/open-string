OpenString is an open-source temperature string — a cable with temperature
sensors distributed along its length, terminated in a microcontroller that
logs and reports readings. Build one to monitor lake thermoclines, soil
temperature profiles, greenhouse gradients, or any environment where you
need temperature at several points along a line.

Every piece you need is open:

- Hardware design files (CERN-OHL-S 2.0)
- Arduino firmware (GPL-3.0)
- This documentation (CC-BY-SA-4.0)
