# Firmware

OpenString firmware is an Arduino sketch. Source lives in
[`firmware/open_string/`](https://github.com/cameronplanck/open-string/tree/main/firmware).

## 1. Install the Arduino IDE

Download from [arduino.cc](https://www.arduino.cc/en/software) — version
2.x recommended. `arduino-cli` works too if you prefer the command line.

## 3. Install libraries

The required libraries are listed in `firmware/libraries.txt`. Install
each through the Arduino Library Manager, or run:

```bash
arduino-cli lib install <library-name>
```

## 4. Open the sketch

Open `firmware/open_string/open_string.ino` in the Arduino IDE.
